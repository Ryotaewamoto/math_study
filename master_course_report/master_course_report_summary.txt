漸近関数は、最適化問題の解を求めるための有力なツールの一つである。この概念の起源は、1913年の Steinitz にあり、その後、 Stocker によって研究が進められた。その20年後、Bourbaki と Choquet によってさらに研究が進められた。1970年、Rockafellar によって凸集合に対しての asymptotic cone を recession cone として定義された。同様に、asymptotic function も recession function として定義され、この理論の基礎となる理論がまとめられた。1999年には、Auslender と Teboulle はこの asymptotic cone と asymptotic function の理論と応用を『Asymptotic cones and functions in optimization and variational inequalities』でまとめた。この本にも書かれているように、漸近関数の応用の一つとして、半正定値計画問題との関係が挙げられる。半正定値計画問題は、線形計画問題、2次凸計画問題、二次錐計画問題の一般化であり、その応用は、組合せ最適化、グラフ理論、統計学、量子力学、機械学習など多岐にわたる。そのため、様々な問題に対して適応できることから近年注目を集めている。
本論文では、漸近関数 (asymptotic function) の半正定値計画問題 (SDP) への応用についてまとめたものである。第1章では、本研究の歴史的背景や意義、概要等について述べる。第2章では、凸関数の定義や性質、その他、凸解析学において重要な概念について述べる。第3章では、漸近集合の定義といくつかの性質について述べ、漸近関数の定義と関連する定理を紹介する。特に、3.1節では漸近集合の定義とその特徴、また様々な表現方法を示す。3.2節では、漸近関数の汎用的な表現方法と共役関数、支持関数との関係を示す。第4章では、漸近関数の半正定値計画問題への応用について、4.1節でこの問題を考える上で鍵となる spectrally defined matrix function を定義する。その後、4.2節でいくつかの論文の結果を紹介する。第5章では、本研究のまとめと今後の課題について述べる。
